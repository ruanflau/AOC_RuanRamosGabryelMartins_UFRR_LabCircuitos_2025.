## [COMPONENTE 08] Somador de 8 bits

### Relatório

#### (i) Descrição
- **Funcionalidade:** Soma `A + B + Cin`.  
- **Entradas:** `A[7..0]`, `B[7..0]`, `Cin`  
- **Saídas:** `S[7..0]`, `Cout`

#### (iii) e (iv) Testes
- **Teste 1:** `A=00010000`, `B=00100001`, `Cin=0` → `S=00110001`  
- **Teste 2:** `Cin=1` → `S=00110010`
