## [COMPONENTE 03] Porta Lógica XOR (usando AND, NOT e OR)

### Relatório

#### (i) Descrição
- **Funcionalidade:** Implementa a operação `Y = (A AND NOT B) OR (NOT A AND B)`.  
- **Entradas:** `A`, `B`  
- **Saída:** `Y`  

#### (iii) e (iv) Testes do Componente
- **Teste 1:** `A=1`, `B=0` → `Y=1`  
- **Teste 2:** `A=1`, `B=1` → `Y=0`
