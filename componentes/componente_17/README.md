## [COMPONENTE 17] Detector de Número Primo

### Relatório

#### (i) Descrição
- **Entrada:** `In[3..0]`  
- **Saída:** `Is_Prime`  
- **Lógica:** Expressão booleana (Mapa de Karnaugh) verdadeira apenas para números primos.

#### (iii) e (iv) Testes
- **Primo:** `In=0111` (7) → `Is_Prime=1`  
- **Não Primo:** `In=1001` (9) → `Is_Prime=0`
